# User Stories
---

## 인증

### US-00a: Google 소셜 로그인
- **사용자로서**, Google 계정으로 간편하게 로그인할 수 있다.
- **그래서**, 별도 회원가입 없이 빠르게 시작할 수 있다.
- **인수 조건**:
  - [ ] "Google로 로그인" 버튼이 있다
  - [ ] 클릭 시 Google OAuth 화면으로 이동한다
  - [ ] 인증 완료 후 앱으로 자동 리다이렉트된다 (온보딩 또는 대시보드)

### US-00b: 로그아웃
- **사용자로서**, 로그아웃할 수 있다.
- **그래서**, 공용 기기에서도 안전하게 사용할 수 있다.
- **인수 조건**:
  - [ ] 설정 페이지에 로그아웃 버튼이 있다
  - [ ] 로그아웃 후 로그인 페이지로 이동한다

---

## 온보딩 (학습 목표 설정)

### US-01: 학습 과목 입력
- **사용자로서**, 내가 공부할 과목들을 입력할 수 있다.
- **그래서**, 과목별로 체계적인 학습 계획을 받을 수 있다.
- **인수 조건**:
  - [ ] 과목명을 텍스트로 입력할 수 있다
  - [ ] 여러 과목을 추가/삭제할 수 있다
  - [ ] 최소 1개 이상의 과목이 있어야 다음 단계로 진행된다

### US-02: 목표 날짜 설정
- **사용자로서**, 학습 목표 날짜(시험일, 마감일)를 설정할 수 있다.
- **그래서**, 남은 기간에 맞춰 일정이 자동으로 분배된다.
- **인수 조건**:
  - [ ] 날짜 선택기(date picker)로 목표일을 선택할 수 있다
  - [ ] 오늘 이전 날짜는 선택할 수 없다
  - [ ] 선택한 날짜까지 남은 일수가 표시된다

### US-03: 하루 가용 학습 시간 입력
- **사용자로서**, 하루에 공부할 수 있는 시간을 입력할 수 있다.
- **그래서**, 현실적인 학습 계획을 받을 수 있다.
- **인수 조건**:
  - [ ] 시간 단위로 입력할 수 있다 (예: 3시간)
  - [ ] 최소 30분 ~ 최대 16시간 범위로 제한된다

---

## 학습 계획 자동 생성

### US-04: 일별 학습 스케줄 생성
- **사용자로서**, 입력한 정보를 기반으로 자동 생성된 일별 학습 계획을 받을 수 있다.
- **그래서**, 매일 무엇을 공부할지 고민하지 않아도 된다.
- **인수 조건**:
  - [ ] 목표일까지의 모든 날짜에 학습 항목이 배정된다
  - [ ] 과목별 시간이 균등하게 배분된다
  - [ ] 생성된 계획이 캘린더 형태로 표시된다

### US-05: 과목별 시간 비중 조절
- **사용자로서**, 특정 과목에 더 많은 시간을 배분할 수 있다.
- **그래서**, 취약 과목에 집중할 수 있다.
- **인수 조건**:
  - [ ] 과목별 비중을 조절할 수 있다 (예: 수학 50%, 영어 30%, 코딩 20%)
  - [ ] 비중 변경 시 스케줄이 자동으로 재계산된다

---

## 진행 관리

### US-06: 오늘의 학습 체크
- **사용자로서**, 오늘 할 학습 항목을 완료 체크할 수 있다.
- **그래서**, 오늘 할 일을 명확히 파악하고 완료 여부를 기록할 수 있다.
- **인수 조건**:
  - [ ] 각 학습 항목에 체크박스가 있다
  - [ ] 체크 시 완료 상태로 변경되고 시각적으로 구분된다
  - [ ] 체크 상태가 LocalStorage에 저장된다

### US-07: 전체 진행률 확인
- **사용자로서**, 전체 학습 진행률을 한눈에 볼 수 있다.
- **그래서**, 현재 얼마나 진행했는지 동기부여를 받을 수 있다.
- **인수 조건**:
  - [ ] 프로그레스 바로 전체 진행률(%)이 표시된다
  - [ ] 완료한 항목 수 / 전체 항목 수가 표시된다

### US-08: 미완료 항목 자동 이월
- **사용자로서**, 어제 못한 학습이 오늘로 자동 이월된다.
- **그래서**, 놓친 학습을 잊지 않고 다시 할 수 있다.
- **인수 조건**:
  - [ ] 지난 날짜의 미완료 항목이 오늘 할 일에 추가된다
  - [ ] 이월된 항목은 시각적으로 구분된다 (예: 색상, 라벨)

---

## 대시보드

### US-09: 오늘의 학습 요약
- **사용자로서**, 앱을 열면 오늘 해야 할 학습이 바로 보인다.
- **그래서**, 빠르게 오늘의 할 일을 파악하고 시작할 수 있다.
- **인수 조건**:
  - [ ] 대시보드가 기본 랜딩 페이지이다
  - [ ] 오늘의 과목, 학습 시간이 카드 형태로 표시된다
  - [ ] 이월된 항목이 있으면 별도로 표시된다

### US-10: 주간 진행률 확인
- **사용자로서**, 이번 주 학습 달성률을 확인할 수 있다.
- **그래서**, 주간 단위로 학습 페이스를 점검할 수 있다.
- **인수 조건**:
  - [ ] 월~일 기준 주간 달성률이 표시된다
  - [ ] 요일별 완료/미완료가 시각적으로 구분된다

### US-11: 과목별 달성률 시각화
- **사용자로서**, 과목별로 얼마나 진행했는지 볼 수 있다.
- **그래서**, 어떤 과목이 부족한지 파악할 수 있다.
- **인수 조건**:
  - [ ] 과목별 프로그레스 바 또는 차트가 표시된다
  - [ ] 각 과목의 완료율(%)이 숫자로 표시된다

---

## 설정

### US-12: 과목 추가/삭제
- **사용자로서**, 학습 중간에 과목을 추가하거나 삭제할 수 있다.
- **그래서**, 계획을 유연하게 변경할 수 있다.
- **인수 조건**:
  - [ ] 설정 페이지에서 과목을 추가/삭제할 수 있다
  - [ ] 변경 시 남은 기간의 스케줄이 자동으로 재계산된다

### US-13: 가용 시간 수정
- **사용자로서**, 하루 가용 학습 시간을 변경할 수 있다.
- **그래서**, 상황 변화에 맞게 계획을 조정할 수 있다.
- **인수 조건**:
  - [ ] 설정 페이지에서 가용 시간을 수정할 수 있다
  - [ ] 변경 시 남은 기간의 스케줄이 자동으로 재계산된다

---

## 데이터 저장

### US-14: 데이터 유지
- **사용자로서**, 브라우저를 닫았다 다시 열어도 데이터가 유지된다.
- **그래서**, 매번 다시 입력하지 않아도 된다.
- **인수 조건**:
  - [ ] 모든 데이터(과목, 계획, 진행 상태)가 LocalStorage에 저장된다
  - [ ] 페이지 새로고침/재방문 시 이전 상태가 복원된다

### US-15: 데이터 초기화
- **사용자로서**, 모든 데이터를 초기화하고 처음부터 다시 시작할 수 있다.
- **그래서**, 새로운 시험이나 목표에 맞게 리셋할 수 있다.
- **인수 조건**:
  - [ ] 설정 페이지에 "초기화" 버튼이 있다
  - [ ] 초기화 전 확인 모달이 표시된다
  - [ ] 초기화 후 온보딩 페이지로 이동한다
