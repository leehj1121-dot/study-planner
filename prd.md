# 학습 플래닝 웹앱 (MVP)

## 1. 제품 개요

개인별 학습 목표와 일정에 맞춰 학습 계획을 생성하고 관리할 수 있는 웹 애플리케이션.

## 2. 목표 사용자

- 시험/자격증을 준비하는 학생 및 직장인
- 스스로 학습 일정을 관리하기 어려운 사람

## 3. 핵심 문제

- 학습 계획을 세워도 실천하기 어렵다
- 과목/분량 배분을 어떻게 해야 할지 모른다
- 진행 상황을 한눈에 파악하기 어렵다

## 4. MVP 핵심 기능

### 4.0 인증
- Google 소셜 로그인 (Supabase Auth OAuth)
- 로그아웃

### 4.1 학습 목표 설정
- 학습 과목 입력 (예: 수학, 영어, 코딩)
- 목표 날짜 설정 (시험일, 마감일 등)
- 하루 가용 학습 시간 입력

### 4.2 학습 계획 자동 생성
- 입력된 정보를 기반으로 일별 학습 스케줄 생성
- 과목별 시간 배분 자동 계산
- 캘린더/타임라인 형태로 계획 표시

### 4.3 진행 관리
- 일별 체크리스트 (완료/미완료 체크)
- 전체 진행률 표시 (프로그레스 바)
- 미완료 항목 자동 이월

### 4.4 대시보드
- 오늘의 학습 할 일 요약
- 주간/월간 진행률 한눈에 보기
- 과목별 달성률 시각화

## 5. MVP 제외 기능 (향후 고려)

- AI 기반 맞춤 추천 (취약 과목 분석 등)
- 소셜 기능 (스터디 그룹, 랭킹)
- 알림/푸시 기능
- 모바일 네이티브 앱
- 외부 캘린더 연동 (Google Calendar 등)

## 6. 기술 스택

| 구분 | 기술 |
|------|------|
| 프레임워크 | Next.js 16 (App Router, Turbopack) |
| 언어 | TypeScript |
| UI | Tailwind CSS v4 + shadcn/ui (Radix) |
| 아이콘 | Lucide React |
| 백엔드 API | Next.js Route Handlers (API Routes) |
| 데이터베이스 | Supabase (PostgreSQL) |
| 인증 | Supabase Auth (Google OAuth) |
| 패키지 매니저 | pnpm |
| 배포 | Vercel |

## 7. 화면 구성

| 페이지 | 설명 |
|--------|------|
| 로그인 | Google 소셜 로그인 |
| 온보딩 | 과목, 목표일, 가용시간 입력 폼 |
| 대시보드 | 오늘의 할 일 + 전체 진행률 |
| 캘린더 뷰 | 일별 학습 계획 타임라인 |
| 설정 | 과목 추가/삭제, 시간 수정 |

## 8. 성공 지표

- 사용자가 계획 생성까지 3분 이내 완료
- 일주일 이상 지속 사용률 추적
