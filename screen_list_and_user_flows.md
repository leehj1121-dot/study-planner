# Screen List & User Flows
---

## 1. 화면 목록 (Screen List)

### S-00: 로그인 / 회원가입
| 항목 | 내용 |
|------|------|
| 경로 | `/login` |
| 관련 스토리 | US-00a |
| 주요 요소 | "Google로 로그인" 버튼 |
| 진입 조건 | 미로그인 상태로 앱 접근 시 |
| 이탈 조건 | 인증 성공 시 → 온보딩 미완료면 S-01, 완료면 S-04로 이동 |

### S-01: 온보딩 - 과목 입력
| 항목 | 내용 |
|------|------|
| 경로 | `/onboarding/subjects` |
| 관련 스토리 | US-01 |
| 주요 요소 | 과목명 텍스트 입력, 추가(+) 버튼, 과목 리스트(삭제 가능), "다음" 버튼 |
| 진입 조건 | 최초 방문 또는 데이터 초기화 후 |
| 이탈 조건 | 과목 1개 이상 입력 시 "다음" 버튼 활성화 |

### S-02: 온보딩 - 목표 날짜 설정
| 항목 | 내용 |
|------|------|
| 경로 | `/onboarding/date` |
| 관련 스토리 | US-02 |
| 주요 요소 | 날짜 선택기(date picker), 남은 일수 표시, "이전"/"다음" 버튼 |
| 진입 조건 | S-01에서 "다음" 클릭 |
| 이탈 조건 | 유효한 날짜 선택 시 "다음" 버튼 활성화 |

### S-03: 온보딩 - 가용 시간 & 비중 설정
| 항목 | 내용 |
|------|------|
| 경로 | `/onboarding/time` |
| 관련 스토리 | US-03, US-05 |
| 주요 요소 | 하루 가용 시간 슬라이더/입력, 과목별 비중 슬라이더(%), 비중 합계 표시, "계획 생성" 버튼 |
| 진입 조건 | S-02에서 "다음" 클릭 |
| 이탈 조건 | 시간 입력 + 비중 합계 100% 시 "계획 생성" 버튼 활성화 |

### S-04: 대시보드 (메인)
| 항목 | 내용 |
|------|------|
| 경로 | `/dashboard` |
| 관련 스토리 | US-06, US-07, US-08, US-09, US-10, US-11 |
| 주요 요소 | 오늘의 학습 카드(체크박스 포함), 이월 항목 영역, 전체 진행률 프로그레스 바, 주간 달성률 차트, 과목별 달성률 바, 하단 네비게이션 |
| 진입 조건 | 온보딩 완료 후 또는 재방문 시 기본 랜딩 |

### S-05: 캘린더 뷰
| 항목 | 내용 |
|------|------|
| 경로 | `/calendar` |
| 관련 스토리 | US-04 |
| 주요 요소 | 월간 캘린더 그리드, 날짜별 과목 태그, 날짜 클릭 시 상세 일정 패널, 오늘 날짜 강조 |
| 진입 조건 | 하단 네비게이션에서 "캘린더" 탭 클릭 |

### S-06: 설정
| 항목 | 내용 |
|------|------|
| 경로 | `/settings` |
| 관련 스토리 | US-12, US-13, US-15 |
| 주요 요소 | 과목 리스트(추가/삭제), 과목별 비중 조절, 가용 시간 수정 입력, "변경 저장" 버튼, "데이터 초기화" 버튼 |
| 진입 조건 | 하단 네비게이션에서 "설정" 탭 클릭 |

### S-07: 초기화 확인 모달
| 항목 | 내용 |
|------|------|
| 관련 스토리 | US-15 |
| 주요 요소 | 경고 메시지, "초기화" 확인 버튼, "취소" 버튼 |
| 진입 조건 | S-06에서 "데이터 초기화" 클릭 |

## 2. 유저 플로우 (User Flows)

### Flow 1: 최초 사용 - 로그인 → 온보딩 → 계획 생성

```
[앱 첫 방문]
    │
    ▼
┌──────────────────────────┐
│  S-00: 로그인 / 회원가입   │
│  Google 로그인              │
└─────────┬────────────────┘
          │ 인증 성공
          ▼
┌─────────────────────┐
│  S-01: 과목 입력     │
│  과목명 입력 → 추가   │
│  (최소 1개 이상)      │
└─────────┬───────────┘
          │ "다음" 클릭
          ▼
┌─────────────────────┐
│  S-02: 목표 날짜     │
│  date picker로 선택   │
│  남은 일수 확인       │
└─────────┬───────────┘
          │ "다음" 클릭
          ▼
┌─────────────────────┐
│  S-03: 시간 & 비중   │
│  하루 가용 시간 입력   │
│  과목별 비중(%) 조절   │
└─────────┬───────────┘
          │ "계획 생성" 클릭
          ▼
┌─────────────────────┐
│  S-04: 대시보드      │
│  오늘의 학습 표시     │
│  계획 생성 완료!      │
└─────────────────────┘
```

### Flow 2: 일상 사용 - 오늘의 학습 수행

```
[앱 재방문]
    │
    ▼
┌──────────────────────────┐
│  S-04: 대시보드            │
│  오늘의 학습 카드 확인      │
│  (이월 항목 있으면 표시)    │
└─────────┬────────────────┘
          │
          ▼
┌──────────────────────────┐
│  학습 항목 체크            │
│  체크박스 클릭 → 완료 처리  │
│  진행률 바 실시간 업데이트   │
└─────────┬────────────────┘
          │ 모두 완료
          ▼
┌──────────────────────────┐
│  오늘 학습 완료!           │
│  전체 진행률 반영           │
│  주간 달성률 업데이트       │
└──────────────────────────┘
```

### Flow 3: 캘린더에서 계획 확인

```
[대시보드]
    │
    │ 하단 네비 "캘린더" 탭
    ▼
┌──────────────────────────┐
│  S-05: 캘린더 뷰          │
│  월간 캘린더에 과목 태그    │
│  오늘 날짜 강조            │
└─────────┬────────────────┘
          │ 특정 날짜 클릭
          ▼
┌──────────────────────────┐
│  해당 날짜 상세 패널       │
│  - 과목별 학습 시간        │
│  - 완료/미완료 상태        │
└──────────────────────────┘
```

### Flow 4: 설정 변경

```
[대시보드]
    │
    │ 하단 네비 "설정" 탭
    ▼
┌──────────────────────────┐
│  S-06: 설정               │
│  과목 추가/삭제            │
│  비중 조절                │
│  가용 시간 수정            │
└─────────┬────────────────┘
          │ "변경 저장" 클릭
          ▼
┌──────────────────────────┐
│  스케줄 자동 재계산        │
│  대시보드로 이동           │
│  변경된 계획 반영          │
└──────────────────────────┘
```

### Flow 5: 데이터 초기화

```
[설정 페이지]
    │
    │ "데이터 초기화" 클릭
    ▼
┌──────────────────────────┐
│  S-07: 초기화 확인 모달    │
│  "모든 데이터가 삭제됩니다" │
├──────────┬───────────────┘
│          │
▼          ▼
"취소"    "초기화"
→ 설정     → 서버 데이터 삭제
  으로       → S-01 온보딩으로
  복귀         이동
```

```
                    ┌──────────┐
                    │ 최초 방문  │

                         ▼
              ┌─────────────────────┐
              │  S-00: 로그인        │
              │  Google 로그인       │
              └─────────┬───────────┘
                        │ 인증 성공
                        ▼
              ┌─────────────────────┐

              └─────────┬───────────┘
                        ▼
         ┌──────────────────────────────┐
         │                              │
    ┌────┴─────┐  ┌──────────┐  ┌──────┴───┐
    │ S-04     │  │ S-05     │  │ S-06     │
    │ 대시보드  │◄►│ 캘린더   │◄►│ 설정     │

    └──────────┘  └──────────┘  └────┬─────┘
                                     │
                              ┌──────┴───────┐
```
